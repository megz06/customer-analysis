# Customer Analysis

## About This Project
Analysis of customer segments and region using 9,994 sales records from a US retail store.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Findings
1. Consumer segment has highest sales
2. West region is the strongest market
3. South region has lowest sales
4. SM-20320 is the top customer by sales

## Skills Demonstrated
- Data loading and exploration
- Groupby analysis
- Cuatomer segmentation
- Data Visualization (Bar chart)
